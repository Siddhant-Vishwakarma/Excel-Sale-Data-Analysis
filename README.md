# Excel-Sale-Data-Analysis(Excel)

Excel based sales data analysis dashboard with insight on profitability,trends,state by performance,category wise performance 

# Dataset 
The Dataset contains order-level sales information including:
- Order ID,Order Date,Year,Month,
- Ship Date,Ship Mode,Customer ID,
- Customer Name,Segment,Country,City
- State,Postal Code,Region,Product ID,
- Category,Sub-Category,Product Name,Sales,Quantity,Discount,Profit

# Business Questions Answered
- Which State generates high Sales?
- Which product generates high sales?
- Top 5 Customers Making high profit?
- Year wise Customer Counts?
- Analyze Sales by Month?
- Find Year wise trends?
  
# Data Cleaning & Preparation
- Removed duplicate records
- Converted text dates into proper date format
- Extracted Year and Month from Order Date
- Handled blank and inconsistent values
